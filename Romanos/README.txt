INTRUCCIONES DE USO
1- Inciar la consola de windows (Presionar la tecla de windows + R una vez abierta la venta de ejecutar, escribriremos cmd y daremos enter)
2- Una vez abierta la consola deberemos acceder la carpeta donde se encuentra el programa (cd desktop/Romanos)
3- Ejecutaremos los comandos de java para poder compilar nuestro programa y que nos genere nuestro archivo class el comando es el siguiente: javac Calculadora_Romanos.java
4- una vez compilado y sin errores procederemos a ejecutar el programa con el comando java Calculadora_Romanos
5- en la consola se nos desplegara una serie de instrucciones las cuales debemos seguir para su correcto uso
6- nos desplegara un mensaje para ingresar los valores necesarios para el proceso de conversion
7- Nos mostrara en pantalla la conversion realizada asi como el valor que ingresamos.
6- Fin.
