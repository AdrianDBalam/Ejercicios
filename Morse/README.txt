INTRUCCIONES DE USO
1- Inciar la consola de windows (Presionar la tecla de windows + R una vez abierta la venta de ejecutar, escribriremos cmd y daremos enter)
2- Una vez abierta la consola deberemos acceder la carpeta donde se encuentra el programa (cd desktop/Morse)
3- Ejecutaremos los comandos de java para poder compilar nuestro programa y que nos genere nuestro archivo class el comando es el siguiente: javac Morse.java
4- una vez compilado y sin errores procederemos a ejecutar el programa con el comando java Morse
5- en la consola se nos desplegara una serie de instrucciones las cuales debemos seguir para su correcto uso
6- Fin.
