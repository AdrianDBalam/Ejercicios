INTRUCCIONES DE USO
1- Inciar la consola de windows (Presionar la tecla de windows + R una vez abierta la venta de ejecutar, escribriremos cmd y daremos enter)
2- Una vez abierta la consola deberemos acceder la carpeta donde se encuentra el programa (cd desktop/Objetos)
3- Ejecutaremos los comandos de java para poder compilar nuestro programa y que nos genere nuestro archivo class el comando es el siguiente: javac Herencia_Objetos.java
4- una vez compilado y sin errores procederemos a ejecutar el programa con el comando java Herencia_Objetos
5- En la consola nos desplegara una serie de opciones las cuales mediante numeros se pueden seleccionar
Las opciones son 1- seleccionar cuadrado 2- seleccionar cubo 3- salir
6- dentro de las 2 primeras opciones se encuentra un submenu en el cual nos dara acceso a diversar funciones para cada una de las figuras
en el caso de cuadrado podremos calcular area y perimetro, mientras que en cubo podremos calcular volumen y perimetro y ambas cuentan con la opcion de regresar 
al menu principal.
7- una vez seleccionada la figura y la operacion desea se nos pedira ingresar el valor necesario para la operacion.
8- nos mostrara el resultado en pantalla.
9- el programa no finalizara hasta seleccionar la opcion 3 en el menu principal.
9- Fin.
